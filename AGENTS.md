# AGENTS

## Git workflow

- Commit messages must follow Conventional Commits (e.g., `feat:`, `fix:`, `chore:`, `docs:`, `refactor:`, `test:`, `build:`, `ci:`).

## Package workflow

- Run `npm run check` before release-related changes.
- Do not commit generated `.runs/` outputs or packed `.tgz` files.
- Keep model-facing prompts free of competitive framing; use neutral terms such as `additional drafts`, `alternative versions`, and `candidate answers`.
